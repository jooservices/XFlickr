# Flickr accounts

Path: `/flickr/accounts`

Manage connected Flickr accounts and run account-level operations.

## Account list

Shows each connected account with:

- Display name and username
- Active status (one account is active at a time)
- API quota meter
- Links to Contacts and Photos catalog

## Actions per account

| Action | Description |
|---|---|
| **Crawl** | Index contacts, photos, photosets, galleries, or favorites for this account |
| **Download** | Queue download jobs for all pending photos owned by this account |
| **Upload** | Queue upload jobs for locally stored photos to default storage |

## Connect / disconnect

- Connect new accounts: **Settings** → Flickr tab → Connect Flickr
- Disconnect: Settings → Flickr tab
- Switch active account: click **Set active** on an inactive account row
